# Clustering

Este directorio contiene un modelo de clustering diseñado para analizar el comportamiento de los clientes basado en:

- **País**: Identificar patrones de comportamiento según la ubicación geográfica.
- **Edad**: Agrupar clientes en función de sus rangos de edad.
- **Género**: Analizar diferencias en el comportamiento según el género.

El objetivo es segmentar a los clientes en grupos homogéneos para entender mejor sus características y necesidades.
