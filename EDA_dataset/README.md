# Dataset y Proceso de Limpieza

Este directorio contiene:

1. **Dataset limpio**: Una versión procesada y lista para usar del conjunto de datos original.
2. **Proceso de limpieza**: Documentación y scripts utilizados para limpiar y preparar los datos. Esto incluye:
   - Manejo de valores nulos.
   - Normalización y estandarización de datos.
   - Eliminación de datos duplicados o irrelevantes.

El objetivo es garantizar que los datos sean de alta calidad y adecuados para el análisis y modelado.
