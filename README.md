# Proyecto: Análisis del Comportamiento de Clientes en E-commerce

Este proyecto tiene como objetivo analizar el comportamiento de los clientes en un entorno de comercio electrónico, utilizando técnicas de análisis de datos y aprendizaje automático.

Haremos uso del dataset de Kaggle "E-Commerce Behavior Data from Multi Category Store" disponible en: https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store

## Estructura del Proyecto

- **EDA_dataset/**: Contiene el dataset limpio y el proceso de limpieza de datos. Se utiliza R para realizar el análisis exploratorio y la limpieza de los datos.
- **clustering/**: Incluye un modelo de clustering desarrollado en Python para segmentar clientes según país, edad y género.
- **prediction/**: Contiene un modelo predictivo en Python para determinar si un cliente realizará una compra o no.

## Tecnologías Utilizadas

- **Python**: Para el desarrollo de modelos de clustering y predicción.
- **R**: Para el análisis exploratorio y la limpieza de datos.

## Objetivos

1. **Segmentación de clientes**: Identificar grupos homogéneos de clientes para personalizar estrategias de marketing.
2. **Predicción de comportamiento**: Anticipar si un cliente realizará una compra para optimizar recursos y estrategias.
3. **Calidad de datos**: Garantizar que los datos utilizados sean precisos y estén listos para el análisis.
